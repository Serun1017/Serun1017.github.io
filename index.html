About Me

"AI와 데이터를 안정적으로 연결하고, 병목 없는 비동기 분산 아키텍처를 설계하는 백엔드 엔지니어"

단순히 API를 개발하는 것을 넘어, 대규모 트래픽 환경에서의 동시성 제어와 고순도 데이터 파이프라인 설계에 강점을 가진 백엔드 엔지니어입니다.

  • 대규모 트래픽과 AI 서빙의 융합: Microsoft Data School 3기를 수료하며 클라우드(Azure) 환경에서 AI 연산망과 웹 서버 간의 트래픽을 통제하는 비동기 시스템을 설계해 2회 연속 프로젝트 1등을 달성했습니다. 다중 접속 시 발생하는 DB 병목을 Redis 분산 락(Lock)과 Pub/Sub 모델, SSE 단방향 스트리밍으로 해결했습니다.
  • FinOps 관점의 비용 최적화: 에이전틱 AI(Agentic AI) 시대에 맞춰 단순히 모델을 연동하는 것에 그치지 않고, 3중 텍스트 정제 파이프라인과 스마트 청킹(Smart Chunking)을 직접 구현해 시스템 토큰 비용을 62% 이상 절감한 경험이 있습니다.
  • ADR(아키텍처 결정 기록) 기반의 소통: 직면한 병목 현상의 원인을 분석하고 구조적으로 해결해 나가는 과정을 80여 편의 기술 블로그에 기록하고 있습니다. 시연 영상과 명확한 기술 문서화를 통해 논리적으로 소통하는 팀워크 지향적 개발자입니다.
